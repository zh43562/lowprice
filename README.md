## 需要的东西：
    一个发件邮箱    
    一个收件邮箱  （可以是同一个）
    我用的QQ邮箱，其他邮箱可能需要改一下发送邮件部分代码

## 需要在Action里加4个参数：
    SENDER_EMAIL  发件邮箱  
    EMAIL_PASSWORD  发件邮箱的授权码
    PROXY_URL  代理ip
    RECEIVER_EMAIL  收件邮箱
